Release Notes for USBPD-TCPP0203 Driver

Purpose

This driver provides an USB Type-C™ Power Delivery interface between STM32G0 or STM32G4 and TCPP03-M20 or TCPP02-M18.

Here is the list of references to user documents:

  • TCPP02-M18 page.
